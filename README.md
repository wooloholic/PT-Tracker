# PT Tracker

A weekly physical therapy exercise tracker built for daily use.

Tracks a full PT program across 7 days — PT days (Mon/Wed/Fri) with the complete exercise sequence, and walk days (Tue/Thu/Sat/Sun) with incline walking. Includes ankle stretch logging, daily notes, and a confetti burst when you finish a day.

## Features

- Check off exercises as you go
- Progress ring per day, updates live
- Confetti on day completion
- Ankle stretch count input per day
- Notes field on every day
- Weekly stats: total done, completion %, days complete, streak
- Auto-saves to browser local storage
- Resets automatically each new week
- Export weekly log as a text file

## How to use

Open `pt-tracker.html` in any browser. No installation, no internet required.

## Program

**Mon / Wed / Fri — PT + walk**
1. Band steps above knee — 1×20
2. Calf stretch — 3×40 sec
3. Step touches — 2×20 (hip motion focus)
4. Band steps around feet — 2×20
5. Dumbbell pass — 2×15 (COIL)
6. Step touch — 2×20
7. Double hop — 3×20 sec
8. SL balance eyes closed — 3×30 sec
9. Ski hop — 3×20 sec
10. SL balance eyes closed — 3×30 sec
11. Tip toe walk — 2×2 laps (10# small steps)
12. Wedge step ups to single calf — 3×25
13. Walk 2 miles — flat
14. Nerve floss — 1×1

**Tue / Thu / Sat / Sun — Walk day**
1. Walk 2 miles + 10 min @ 6% grade
2. Nerve floss — 1×1

All days include ankle stretch logging and a notes field.
